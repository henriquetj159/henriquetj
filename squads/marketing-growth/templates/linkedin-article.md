# Template: LinkedIn Article (Long-Form)

**Specs:** 800-2000 palavras | Imagem capa 1200x644px

## Titulo

Formato: "[Numero] [Promessa especifica] [Para quem]"

## Introducao (150-200 palavras)

```
[Hook - frase de impacto]
[Contexto - por que este tema importa]
[Promessa - o que o leitor vai aprender]
[Credibilidade - por que voce esta qualificado]
```

## Corpo (500-1500 palavras)

```
## [Secao 1 - H2]
[Conteudo com exemplos concretos]

## [Secao 2 - H2]
[Conteudo]

## [Secao 3 - H2]
[Conteudo]
```

## Conclusao (100-200 palavras)

```
## Proximos Passos
[Resumo dos pontos principais]
[1-3 acoes praticas]
[CTA: conectar, comentar, seguir]
```

## SEO

- Keyword no titulo e primeiro paragrafo
- Headings descritivos
- 3-5 hashtags no final
