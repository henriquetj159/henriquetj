# Conversion Elements Checklist

## Headline

- [ ] Promessa clara e especifica?
- [ ] Beneficio principal (nao feature)?
- [ ] Urgencia ou curiosidade?

## Proposta de Valor

- [ ] Valor percebido > preco?
- [ ] Diferencial claro?
- [ ] Beneficios quantificados?

## Social Proof

- [ ] Depoimentos reais (nome + foto + contexto)?
- [ ] Numeros de clientes/resultados?
- [ ] Minimo 3 provas sociais?

## CTA

- [ ] Verbo de acao ("Garantir vaga", "Comecar agora")?
- [ ] Botao visualmente destacado?
- [ ] CTA repetido (hero + meio + final)?
- [ ] Micro-copy sob CTA?

## Remocao de Risco

- [ ] Garantia explicita?
- [ ] Periodo de garantia claro?
- [ ] FAQ cobre objecoes principais?

## Urgencia e Escassez

- [ ] Escassez REAL (nao fabricada)?
- [ ] Deadline claro?

## Formulario

- [ ] Minimo de campos possivel?
- [ ] Labels claros?
- [ ] Botao submit descritivo?
