# Accessibility WCAG AA Checklist

## Contrast

- [ ] Normal text contrast >= 4.5:1?
- [ ] Large text contrast >= 3:1?
- [ ] UI components contrast >= 3:1?
- [ ] Focus indicators visible?

## Semantics

- [ ] Correct heading hierarchy (single H1)?
- [ ] HTML landmarks (header, main, nav, footer)?
- [ ] Lists used for list content?
- [ ] Links vs buttons used correctly?

## ARIA

- [ ] ARIA labels on interactive elements?
- [ ] ARIA landmarks defined?
- [ ] Role attributes where needed?
- [ ] aria-hidden on decorative elements?

## Keyboard

- [ ] Full tab navigation works?
- [ ] Focus order is logical?
- [ ] No keyboard traps?
- [ ] Skip to content link?

## Images & Media

- [ ] Alt text on all informative images?
- [ ] Decorative images have empty alt=""?
- [ ] Video captions (if applicable)?

## Touch

- [ ] Touch targets minimum 44x44px?
- [ ] Adequate spacing between targets?
