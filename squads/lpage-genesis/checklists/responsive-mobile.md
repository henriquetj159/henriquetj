# Responsive Mobile Checklist

## Viewport

- [ ] Mobile-first approach (375px base)?
- [ ] No horizontal scroll at any breakpoint?
- [ ] Viewport meta tag correct?

## Layout

- [ ] Single column on mobile?
- [ ] Stacked elements (no side-by-side cramping)?
- [ ] Full-width CTAs on mobile?
- [ ] Adequate padding (16px+ sides)?

## Typography

- [ ] Body text minimum 16px on mobile?
- [ ] Headings scale down proportionally?
- [ ] Line length comfortable (45-75 characters)?

## Touch

- [ ] Touch targets minimum 44x44px?
- [ ] Adequate spacing between tap targets (8px+)?
- [ ] No hover-only interactions?

## Images

- [ ] Responsive images (srcset or CSS)?
- [ ] No images wider than viewport?
- [ ] Lazy loading on below-fold images?

## Breakpoints Tested

- [ ] 375px (iPhone SE)?
- [ ] 393px (iPhone 14 Pro)?
- [ ] 820px (iPad)?
- [ ] 1024px (Laptop)?
- [ ] 1440px (Desktop)?
