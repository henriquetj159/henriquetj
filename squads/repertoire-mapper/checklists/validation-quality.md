# Validation Quality Checklist — Repertoire Mapper

## Purpose
Validar consistência, coerência e ausência de contradições críticas antes de entregar os outputs finais.

## Checklist
- [ ] Diagnóstico completo por domínio (tipo + nível por área)
- [ ] Itens categorizados no ciclo SECI (Socialização, Externalização, Combinação, Internalização)
- [ ] Contradições críticas identificadas e resolvidas
- [ ] Gaps de conhecimento explicitados
- [ ] Evidências-chave revisadas (amostra representativa)
- [ ] Conexões cruzadas entre domínios estão coerentes
- [ ] Argyris gap-check executado
- [ ] Validação final aprovada para entrega

## Gate Mapping
- **QG-005 Final Validation** — Bloqueante

## Outcome
- [ ] PASS: Manifesto e manual prontos para entrega
- [ ] FAIL: Reabrir análise e corrigir incoerências
