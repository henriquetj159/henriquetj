## 🎯 **Objetivo**

Integrar Google Sheets MCP para atualizar automaticamente a coluna de status (Coluna H) após cada envio de mensagem, mantendo a planilha sincronizada com o progresso real.

**User Story:**
> Como usuário do Ensinio Prospector, eu quero que o status de envio seja atualizado automaticamente no Google Sheets após cada mensagem enviada, para que eu possa acompanhar o progresso em tempo real sem precisar atualizar manualmente.

## 💡 **Implementation Notes**

### **Why Batch Update?**
- Google Sheets API tem rate limit
- Batch update (10 cells) = 1 API call
- Individual updates (10 cells) = 10 API calls
- Evita throttling

### **Why Queue Pattern?**
- Acumular updates em memória
- Flush quando queue >= 10 ou ao final
- Reduz I/O e API calls

### **Workaround: Apps Script**
Se MCP não estiver disponível, usar webhook:
```javascript
// Apps Script endpoint (deploy as web app)
function doPost(e) {
  const { rowIndex, status } = JSON.parse(e.postData.contents);
  const sheet = SpreadsheetApp.openById('124EQQAkmt9D7-49LbR-Jx64DhxdtCwceUQgqolk5ZFI').getSheetByName('Sheet1');
  sheet.getRange(rowIndex, 8).setValue(status); // Coluna H = 8
  return ContentService.createTextOutput(JSON.stringify({ success: true }));
}
```

Then call from Node.js:
```javascript
async function updateSheetStatus(rowIndex, status) {
  const url = 'https://script.google.com/macros/s/.../exec';
  await fetch(url, {
    method: 'POST',
    body: JSON.stringify({ rowIndex, status }),
  });
}
```
