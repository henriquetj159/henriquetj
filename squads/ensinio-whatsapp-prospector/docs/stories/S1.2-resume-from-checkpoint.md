## 🎯 **Objetivo**

Implementar sistema de checkpoint para salvar progresso de envio e permitir retomar de onde parou sem re-enviar para contatos que já receberam mensagem.

**User Story:**
> Como usuário do Ensinio Prospector, eu quero que o sistema salve automaticamente o progresso do envio, para que eu possa parar a qualquer momento e retomar depois sem perder o progresso ou re-enviar mensagens duplicadas.

- [ ] Função `loadCheckpoint(groupSlug)` carrega JSON do arquivo
- [ ] Se arquivo não existir, retornar `null` (primeira execução)

- [ ] Retornar erro se JSON corrompido

### **AC-4: Resume Logic**
- [ ] Ao iniciar envio, verificar se checkpoint existe
- [ ] Se existir, carregar e retomar do `lastSentIndex + 1`
- [ ] Filtrar apenas contatos NÃO enviados
- [ ] Validar que não re-envia para mesmos índices
- [ ] Atualizar totais acumulados (totalSent, totalErrors)

### **AC-5: CLI Flag**
- [ ] Flag `--resume` para forçar retomar de checkpoint
- [ ] Flag `--restart` para ignorar checkpoint e começar do zero
- [ ] Se `--restart`, deletar checkpoint existente antes de iniciar
- [ ] Avisar usuário se checkpoint existe e nenhuma flag foi passada

## 💡 **Implementation Notes**

### **Why JSON?**
- Fácil de ler/editar manualmente se necessário
- Estruturado (não plain text)
- Compatible com scripts e ferramentas

### **Why Atomic Write?**
- Evitar corrupção se Ctrl+C durante write
- Pattern: write to `.tmp`, then rename (atomic operation)

### **Why Save After Each Send?**
- Garantir que progresso não se perde mesmo se crash
- Overhead mínimo (I/O rápido)

### **Implementation Reference**
```javascript
const fs = require('fs');
const path = require('path');

const CHECKPOINT_DIR = '.aios/sessions';

function saveCheckpoint(groupSlug, data) {
  const filePath = path.join(CHECKPOINT_DIR, `${groupSlug}-checkpoint.json`);
  const tmpPath = `${filePath}.tmp`;

  // Atomic write
  fs.writeFileSync(tmpPath, JSON.stringify(data, null, 2));
  fs.renameSync(tmpPath, filePath);
}

function loadCheckpoint(groupSlug) {
  const filePath = path.join(CHECKPOINT_DIR, `${groupSlug}-checkpoint.json`);

  if (!fs.existsSync(filePath)) return null;

  try {
    const content = fs.readFileSync(filePath, 'utf8');
    return JSON.parse(content);
  } catch (error) {
    console.error('Checkpoint corrompido:', error.message);
    return null;
  }
}

function deleteCheckpoint(groupSlug) {
  const filePath = path.join(CHECKPOINT_DIR, `${groupSlug}-checkpoint.json`);
  if (fs.existsSync(filePath)) {
    fs.unlinkSync(filePath);
  }
}
```
